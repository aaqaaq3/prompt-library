# رفع دقة الصور (Image Upscaling Prompt)

## 📌 المعلومات الأساسية

| الحقل | القيمة |
|-------|--------|
| **الفئة** | محتوى — تحسين الصور |
| **النموذج الموصى به** | Midjourney / DALL-E / Firefly / Topaz |
| **اللغة** | إنجليزية |
| **الإصدار** | v1.0 |
| **تاريخ الإنشاء** | 2026-06-08 |

---

## 🎯 الهدف

برومبت يُضاف لأي صورة لتحسين جودتها وتفاصيلها ورفع دقتها إلى 8K.

---

## 📝 البرومبت

```
Masterpiece, ultra-high resolution, 8k UHD, photorealistic, hyper-detailed textures, sharp focus, cinematic lighting, professional post-processing, intricate details, vibrant colors, realistic skin pores and hair, denoised, high contrast, crisp edges, award-winning photography.
```

---

## ✅ نصائح الاستخدام

- أضف هذا البرومبت في نهاية أي برومبت صورة لتحسين جودتها
- في Midjourney أضف `--uplight` أو `--upbeta` بعده
- يمكن استخدامه مع أي نموذج لتوليد الصور
