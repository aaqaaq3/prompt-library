# 📚 مكتبة البرومبتات الاحترافية

> مجموعة منتقاة من البرومبتات الجاهزة للاستخدام مع نماذج الذكاء الاصطناعي
> بقلم: أبو عبد الله | مدرب ومتخصص في الذكاء الاصطناعي

---

## 🗂️ هيكل المكتبة

```
prompt-library/
├── تعليم/          # برومبتات التعليم والتدريب
├── محتوى/          # إنشاء المحتوى والكتابة الإبداعية
├── بحث/            # البحث والتحليل
├── أعمال/          # الأعمال والإنتاجية
├── عام/            # برومبتات متعددة الاستخدام
└── _template.md    # قالب إنشاء برومبت جديد
```

---

## 🚀 كيفية الاستخدام

1. تصفح المجلد المناسب لاحتياجك
2. افتح ملف البرومبت
3. انسخ النص من قسم `البرومبت`
4. استبدل المتغيرات بين `{{` و `}}` بقيمك
5. الصقه في أي نموذج AI

---

## 📋 الفئات

| الفئة | الوصف | عدد البرومبتات |
|-------|-------|----------------|
| 🎓 تعليم | شرح المفاهيم، تصميم الدروس، التقييم | متجدد |
| ✍️ محتوى | لينكدإن، واتساب، بلوغ، سوشيال ميديا | متجدد |
| 🔍 بحث | تحليل المعلومات، ملخصات، مقارنات | متجدد |
| 💼 أعمال | تقارير، إيميلات، خطط | متجدد |
| 🌐 عام | برومبتات متعددة الأغراض | متجدد |

---

لديك برومبت جيد؟ افتح **Pull Request** أو **Issue** مع البرومبت باستخدام قالب `_template.md`
